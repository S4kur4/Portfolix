# Privacy

Portfolix is designed as a local-first portfolio tracker.

## Local Data

Portfolio records, historical snapshots, app settings, and provider API keys are stored locally on the user's Mac.

## AI Analysis

When the user triggers AI analysis, Portfolix may send portfolio data to the user-configured LLM provider. If online search is enabled, Portfolix may also query the configured search provider.

Users should review their selected providers' privacy policies before enabling AI or online search features.

## No Brokerage Login

Portfolix does not request brokerage usernames, brokerage passwords, cookies, or trading-session tokens.

## Data Export

Data export packages are created locally and exclude AI chat history, analysis reports, API keys, and app settings.
